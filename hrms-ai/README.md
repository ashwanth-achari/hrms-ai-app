# HRMS-AI

HRMS-AI is a Human Resource Management System powered by AI technologies. This project aims to streamline HR processes, enhance decision-making, and improve employee engagement through intelligent automation and data-driven insights.

## Features

- Employee Management
- Recruitment Automation
- Performance Tracking
- Leave Management
- AI-Powered Analytics

## Installation

To get started with HRMS-AI, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/HRMS-AI.git
   ```

2. Navigate to the project directory:
   ```
   cd HRMS-AI
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the application, execute the following command:
```
python src/hrms_ai/main.py
```
